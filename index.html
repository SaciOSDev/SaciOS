Welcome to Sacramento iOS.

This page has been designed to get you started with git and github. The first thing you should do is fork this repository from the link above. This help assumes you already have git installed. If you do not have git installed follow these instructions.

Forks

  1. Click on fork repository.
  2. Once you are at the project's repository page click on the fork link at the top.

  3. Now you need to clone the repository. From the Console (assuming you already installed git) copy and paste the code bellow.

  4. $ git clone git@github.com:yuric/SaciOS.git This will download a copy of SaciOS files to your machine. Now work on it locally and make some awesome changes. More details on forking a repository here

Saving your work

  1. Saving your changes:

  2. $ git add -A to add all files you changed to commit queue

  3. $ git commit -m "message explaining what you changed" This is the actual commit

  4. $ git push origin (or if you are using a remote other than origin replace accordingly) This will push it to your github branch.

Make more changes and repeat steps above until you are satisfied with what you have and are ready to merge back to Master. Master is the result of our collective work. It is the base branch of code that everyone bases their files from and the branch that we will maintain for release to the app store. This branch should have no bugs (sure, right?) and before your changes are accepted your code will be reviewed.
